# 🍽️ Restaurant Data Analysis – BDM Project (IIT Madras)

## 📌 Overview  
This project was developed as part of the **Business Data Management (BDM)** course at **IIT Madras**.  
It involves the **analysis of restaurant data** to extract meaningful insights, identify challenges, and provide **data-driven recommendations**.  

## 🎯 Objectives  
- 📊 Perform detailed analysis of restaurant operations  
- 🔍 Identify problems faced by the restaurant owner  
- 💡 Provide actionable recommendations and suggestions based on insights  
- 🛠️ Propose effective solutions to optimize management  

## 📑 Project Highlights  
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Insights & Visualization  
- Recommendations & Problem-Solving  

## ✅ Outcome  
The project successfully delivered **insights and strategies** to help the restaurant owner improve efficiency, manage resources better, and enhance customer satisfaction.
